This is the API

getStrainByOcpc

Returns a single strain.

URL

https://api.otreeba.com/v1/strains/{ocpc}

HTTP Method

GET

Response Type

Strain

Parameters

  • Path: ocpc String

    OCPC of the strain to return.

getStrains

Returns a paginated list of strains.

URL

https://api.otreeba.com/v1/strains

HTTP Method

GET

Response Type

inline_response_200_11

Parameters

  • Query: page Integer

    Page to be returned.

  • Query: count Integer

    The number of items to return. Default 10. Max 50.

  • Query: sort String

    How to sort the items.
