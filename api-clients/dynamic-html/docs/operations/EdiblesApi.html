This is the API

getEdibleByOcpc

Returns a single edible.

URL

https://api.otreeba.com/v1/edibles/{ocpc}

HTTP Method

GET

Response Type

Edible

Parameters

  • Path: ocpc String

    OCPC of the edible to return.

getEdibles

Returns a paginated list of edibles.

URL

https://api.otreeba.com/v1/edibles

HTTP Method

GET

Response Type

inline_response_200_2

Parameters

  • Query: page Integer

    Page to be returned.

  • Query: count Integer

    The number of items to return. Default 10. Max 50.

  • Query: sort String

    How to sort the items.
