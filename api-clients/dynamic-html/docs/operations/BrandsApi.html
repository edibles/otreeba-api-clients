This is the API

getBrandByOcpc

Returns a single brand.

URL

https://api.otreeba.com/v1/brands/{ocpc}

HTTP Method

GET

Response Type

Brand

Parameters

  • Path: ocpc String

    OCPC of the brand to return.

getBrandEdibles

Returns a paginated list of edibles.

URL

https://api.otreeba.com/v1/brands/{ocpc}/edibles

HTTP Method

GET

Response Type

inline_response_200_8

Parameters

  • Path: ocpc String

    OCPC of the brand to list edibles for.

  • Query: page Integer

    Page to be returned.

  • Query: count Integer

    The number of items to return. Default 10. Max 50.

  • Query: sort String

    How to sort the items.
