This is the API

getSeedCompanies

Returns a paginated list of seed companies.

URL

https://api.otreeba.com/v1/seed-companies

HTTP Method

GET

Response Type

inline_response_200_1

Parameters

  • Query: page Integer

    Page to be returned.

  • Query: count Integer

    The number of items to return. Default 10. Max 50.

  • Query: sort String

    How to sort the items.

getSeedCompanyByOcpc

Returns a single seed company.

URL

https://api.otreeba.com/v1/seed-companies/{ocpc}

HTTP Method

GET

Response Type

SeedCompany

Parameters

  • Path: ocpc String

    OCPC of the seed company to return.

getSeedCompanyStrainsByOcpc

Returns a paginated list of strains for a single seed company.

URL

https://api.otreeba.com/v1/seed-companies/{ocpc}/strains

HTTP Method

GET

Response Type

inline_response_200

Parameters

  • Path: ocpc String

    OCPC of the seed company to return strains for.

  • Query: page Integer

    Page to be returned.

  • Query: count Integer

    The number of items to return. Default 10. Max 50.
